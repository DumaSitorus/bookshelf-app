# Membangun Bookshelf App
**Proyek Akhir: Belajar Membuat Front-End Web untuk Pemula - Coding Camp 2026 Progressive: Full-Stack Web Developer**


## Deskripsi Proyek  
Bookshelf App adalah aplikasi web sederhana untuk menambahkan, mengedit, mencari dan menghapus buku yang akan dan telah selesai dibaca. Proyek yang bertujuan untuk mengimplementasikan materi [Belajar Membuat Front-End Web untuk Pemula](https://www.dicoding.com/academies/315/corridor). Website ini dibangun menggunakan HTML, CSS, dan Javascript dengan fitur-fitur berikut:
* 1| Menggunakan `localStorage` sebagai penyimpanan data buku.
* 2| Menyimpan data buku lengkap (`Judul`, `Author`, `Tahun Terbit`).
* 3| Buku dikelola melalui 2 rak yaitu `Buku Belum Dibaca` dan `Buku Selesai Dibaca`.
* 4| Pengguna dapat `Memindahakan` buku ke rak lain, `Mengedit` dan `Menghapus` buku.
* 5| Pengguna dapat `Mencari` judul buku melalui search bar.
* 6| Website 'Responsif' terhadap ukuran layar
